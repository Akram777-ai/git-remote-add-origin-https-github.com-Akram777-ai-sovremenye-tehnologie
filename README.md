Описание выполнения лабораторной работы №2

Тема: Переменные, типы данных и наследование в языке программирования Java. Цель: Закрепить навыки работы с переменными, изучить области видимости, механизм наследования и переопределения методов.


Выполненные задачи:

Задача 1: Объявлены примитивные переменные (int, double, boolean) и константа с ключевым словом final.


Задача 2: Продемонстрирована инициализация примитивного типа (int) и ссылочного типа (String).


Задача 3: Реализовано разграничение области видимости: поле класса (classField) доступно всему классу, а локальная переменная (localVariable) — только внутри метода main.


Задача 4: Создана иерархия классов, где класс Student наследует поля name и age от базового класса Person, добавляя собственное поле group.


Задача 5: Реализован механизм полиморфизма: метод displayInfo() переопределен в классе Student с использованием super.displayInfo() и вызван через ссылку базового класса Person.
